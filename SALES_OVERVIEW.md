# 活動簽到與管理系統 — 銷售簡介（賣點與差異化功能）

> 本文從 **銷售／提案** 角度整理本系統 **特別值得講的 function**，方便對客戶說明「為什麼選我們」與「和一般報名工具哪裡不同」。

---

## 一句話定位

**從報名、收款、簽到、互動到抽獎與報表，同一套後台搞定；大螢幕與 iPad 現場連動，不需再拼多套工具。**

---

## 核心賣點（客戶聽得懂的差異）

### 1. 現場抽獎「雙螢幕體驗」— 像活動公司等級

| 賣點 | 說明 |
|------|------|
| **iPad 控制 + 大螢幕同步** | 工作人員用 Panel 抽獎，來賓看的大螢幕 **即時** 出結果，不用舉手報號、不用 Excel 抽完再貼。 |
| **Socket 即時同步** | 選獎品、單抽／批量抽、動畫與名單 **同一個 room 即時推送**，現場不卡、不錯頻。 |
| **獎品圖＋庫存連動** | 選獎品時可預覽圖片；中獎自動扣庫存，避免「畫面還有件數但後台已沒貨」的糗況。 |
| **Winner No 可重置** | 誤刪或要補抽時，可 **重置編號邏輯**，下一個號碼接在「現有名單最大號」之後，符合主辦對「序號連續／可補洞」的期待。 |
| **中獎名單可依 FormConfig 自選欄位** | List／Award 頁／Excel 匯出 **同一套欄位設定**，沒有桌號也能改顯示公司、部門、電話等你們表單裡有的欄位。 |

**對銷售話術：**  
「不是只有網頁抽獎機，是 **控台 + 大螢幕 + 庫存 + 名單匯出** 一條龍，活動當天一人用 iPad 就能控全場。」

---

### 2. 報名表單「完全客製」— 不用改程式

| 賣點 | 說明 |
|------|------|
| **後台拖拉式／區塊式設定** | 欄位類型含文字、Email、電話、下拉、單選、多選、textarea 等，**標籤中／英、必填、可見性** 都可設。 |
| **多語言＋URL 語系** | 同一表單支援中／英；可用 `?lang=` 連結直接開指定語系，適合雙語活動。 |
| **動態欄位進 RSVP／簽到／抽獎** | 報名時收的客製欄位會進入來賓主檔，**簽到列表、抽獎名單、匯出** 都能帶出，不用二次 key-in。 |

**對銷售話術：**  
「客戶要收『部門、職稱、飲食』這種欄位，**不用找工程師改 code**，後台設好就上線。」

---

### 3. 付費活動 — 雙閘道、自動入名單

| 賣點 | 說明 |
|------|------|
| **Stripe / Wonder 可切換** | 同一套報名流程，後端依設定走 **Stripe Checkout** 或 **Wonder**，適合香港／本地金流或既有 Stripe 客戶。 |
| **Webhook 自動轉已付＋寫入 RSVP** | 付款完成 **自動** 把來賓寫進活動名單並標記已付，減少人工對帳。 |
| **交易紀錄＋對帳** | Transaction 保留 session／order 與 callback 資料，方便財務與活動後查詢。 |
| **訂單／付款憑證信** | 建立訂單與付款完成可自動寄信（可套模板），降低客服問「有沒有付成功」的負擔。 |

**對銷售話術：**  
「付費講座、付費報名不用再接人工對帳；**付完即入場名單**，現場簽到直接掃。」

---

### 4. 簽到與現場營運 — 專為「入口」設計

| 賣點 | 說明 |
|------|------|
| **QR 掃描簽到** | 來賓帶 QR 或現場掃，狀態即時更新，可搭配 **Printer Bridge** 印標籤。 |
| **Guest List 與 RSVP 分離** | 可先匯入／維護 **預設來賓**，再與實際報名合流，適合邀請制活動。 |
| **批量簽到／批量寄信** | 大會報到、團體處理時可 **批次操作**，節省人力。 |

---

### 5. 互動加值 — 同一平台延伸

| 模組 | 銷售講法 |
|------|----------|
| **Treasure Hunt** | 現場掃點集點，可綁積分，增加展場動線與停留。 |
| **投票** | 簡報／表演即時投票，結果可後台統計。 |
| **積分＋排行榜** | 掃碼加分、排行榜，適合遊戲化活動。 |
| **Badge 設計** | 活動胸章／名牌視覺可後台配置，強化品牌一致。 |

**對銷售話術：**  
「報名與簽到只是入口；同一套還能加 **尋寶、投票、積分**，客單可以從『報名系統』拉到『整場互動方案』。」

---

### 6. 權限與多活動 — 適合代理商／企業多部門

| 賣點 | 說明 |
|------|------|
| **Admin 與活動級權限** | 可只開某活動的 RSVP、抽獎、匯出等，**外包工讀生不必看到全部資料**。 |
| **多活動列表** | 一個後台管多場活動，適合年曆滿檔的 agency。 |

---

## 和「一般報名頁 + Google 表單」比，我們多什麼？

| 維度 | 一般做法 | 本系統 |
|------|----------|--------|
| 付費報名 | 要接金流＋人工對名單 | Webhook **自動入名單** |
| 現場抽獎 | Excel 或獨立小工具 | **iPad 控台 + 大螢幕即時 + 庫存** |
| 欄位 | 表單固定 | **FormConfig 自訂 + 名單／抽獎／匯出同源** |
| 簽到 | 打勾或另一套 APP | **QR + 批量 + 可列印** |
| 活動後 | 多處匯出再合併 | **單一後台報表／Excel** |

---

## 適合推給誰？

- **品牌活動／發佈會**：要體面的大螢幕抽獎 + 來賓資料齊全。  
- **付費課程／研討會**：要線上收款 + 自動入場名單。  
- **展覽／年會**：Guest List + 簽到 + 尋寶／投票加值。  
- **代理商**：多活動、權限切細、同一套重複賣。

---

## 技術一句帶過（給 IT 安心用）

- 後端 Node + Express，MongoDB；即時 Socket.IO。  
- 支付 Stripe／Wonder 可配置；信件可接 SES 等。  
- 表單與名單 **同一數據模型**，減少資料孤島。
