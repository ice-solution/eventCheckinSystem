## 1. 事件管理 (Event Management)

### 基本操作
- ✅ **創建事件** - 設定事件名稱、開始/結束時間、所有者
- ✅ **事件列表** - 依角色顯示可存取事件，支援搜尋篩選
- ✅ **事件詳情** - 查看事件完整資訊及用戶統計
- ✅ **更新事件名稱** - 即時更新事件名稱
- ✅ **刪除事件** - 刪除事件及所有相關資料

### 事件設定
- ✅ **抽獎設置** - 配置抽獎顯示欄位、密碼保護、背景圖片
- ✅ **背景圖片上傳** - 自訂抽獎頁面背景圖片
- ✅ **付費活動設置** - 啟用付費模式、管理票券
- ✅ **郵件發送設置** - 配置歡迎/確認/提醒/感謝郵件的發送方式
- ✅ **歡迎訊息方式** - 選擇 Email / SMS / Both 三種發送方式
- ✅ **附件管理** - 上傳、管理、刪除事件相關文件

## 2. 用戶管理 (RSVP)

### CRUD 操作
- ✅ **新增用戶** - 手動填寫動態表單欄位新增用戶
- ✅ **批量匯入** - 透過 Excel 檔案批量匯入用戶
- ✅ **下載匯入範本** - 下載含正確欄位的 Excel 範本
- ✅ **用戶列表** - DataTables 顯示所有用戶，支援即時搜尋
- ✅ **用戶詳情** - 查看單一用戶所有欄位及狀態
- ✅ **內聯編輯** - 雙擊表格單元格直接編輯，無需開啟編輯頁
- ✅ **更新用戶** - 更新所有動態表單欄位資料
- ✅ **刪除用戶** - 從事件移除單一用戶
- ✅ **批量刪除** - 勾選多個用戶一鍵刪除

### 用戶功能
- ✅ **QR Code 生成** - 每位用戶自動生成唯一 QR Code
- ✅ **QR Code 下載** - 下載用戶專屬 QR Code 圖片
- ✅ **用戶登入頁面** - 事件專屬登入頁，用戶自助查看資料
- ✅ **QR Code 登入** - 掃描 QR Code 快速登入
- ✅ **用戶資料自助編輯** - 用戶登入後可編輯自己的資料
- ✅ **重新發送郵件** - 為單一用戶重發 Invitation / Welcome / Knowledge / Reminder 郵件

### 批量通訊
- ✅ **批量發送郵件** - 勾選多位用戶批量發送指定郵件模板
- ✅ **批量發送 SMS** - 勾選多位用戶批量發送 SMS 模板

### 統計顯示
- ✅ **簽到統計** - 即時顯示已簽到人數 / 總人數 / 簽到率
- ✅ **積分顯示** - 在用戶列表直接顯示各用戶積分

- ✅ **欄位驗證規則** - 設定最小/最大長度、正則表達式驗證
- ✅ **選項管理** - 為 select/checkbox/radio 設定選項（含「其他」自由填寫項）
- ✅ **欄位可見性** - 個別控制每個欄位的顯示/隱藏
- ✅ **即時預覽** - 配置時即時預覽表單效果

### 表單管理
- ✅ **配置匯出** - 將表單配置匯出為 JSON
- ✅ **配置匯入** - 從 JSON 檔案匯入表單配置
- ✅ **重置為預設** - 一鍵重置為系統預設配置
- ✅ **開關報名頁** - 可關閉報名頁，顯示自訂關閉訊息
- ✅ **預設語言設定** - 設定表單預設顯示語言
